# Extension Logo Assets

Place your extension icon files in this directory.

## Required Files for Chrome Extension

- `icon16.png` - 16x16px (toolbar)
- `icon32.png` - 32x32px (toolbar retina)
- `icon48.png` - 48x48px (extension management)
- `icon128.png` - 128x128px (Chrome Web Store)

## Optional
- `icon.svg` - Vector source file

## Current Files
- Placeholder icons in `/extension/icons/`
- Replace with your branded icons
- Run `npm run sync-branding` to update manifest
